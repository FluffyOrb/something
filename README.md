## Overview
Something is an extensive Angular app demo including:
- responsive, mobile-ready layout
- theming, light and dark modes
- various features to play with

## Details
Something is based on:
- Angular 12
- PrimeNG component library
- Font Awesome icon library

## How to run
Run `npm run start` to start the development server.\
Run `npm run build` to build output bundle.\
Run `npm run analyzeBundle` to analyze output bundle size.\
